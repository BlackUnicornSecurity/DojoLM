# BlackUnicorn Company Assets

Place company-level branding assets here.

## Required Files

| File | Specs | Purpose |
|------|-------|---------|
| `logo.png` | 512x512px min, transparent BG | Main logo for watermarks |
| `logo.svg` | Vector format | Scalable for web/fixtures |
| `favicon.ico` | 32x32px | Web page favicons |
| `watermark.png` | Semi-transparent PNG | Image overlay |
| `audio-intro.mp3` | 2-3 seconds | Audio branding for MP3s |
| `audio-intro.wav` | 2-3 seconds | Audio branding for WAVs |

## Brand Colors (BlackUnicorn)

- Primary: `#000000` (Black)
- Secondary: `#0066CC` (Blue)
- Accent: `#00AAFF` (Light Blue)
