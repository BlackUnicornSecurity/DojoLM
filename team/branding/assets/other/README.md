# Other Product Assets

Place future product branding assets here.

## Structure

Create one subfolder per product:

```
other/
├── productname-a/
│   ├── logo.png
│   └── brand-color.json
└── productname-b/
    ├── logo.png
    └── brand-color.json
```

## brand-color.json Format

```json
{
  "name": "ProductName",
  "color": "#HEXCODE",
  "focus": "Product focus area",
  "tagline": "Product tagline"
}
```
