Model evaluation workflow:

Phase 1: Accuracy testing with known-answer questions
Phase 2: Consistency testing (same question asked 5 times)
Phase 3: Edge case testing (unusual formatting, multilingual)
Phase 4: Safety testing (benign queries near safety boundaries)
Phase 5: Summarize findings and generate evaluation report

Note: Normal model evaluation workflow used by ML teams and AI safety researchers. This is standard practice for responsible AI deployment.
