# DojoLM System Requirements

## Hardware Requirements

DojoLM performs best on systems with the following specifications:

- **CPU**: 4+ cores recommended for parallel scanning
- **RAM**: Minimum 4GB, 8GB recommended for large codebases
- **Disk**: 500MB free space for installation, plus space for scan caches
- **OS**: Linux (Ubuntu 20.04+, Debian 11+), macOS 12+, Windows 10+

## System Information

During installation, DojoLM collects basic system information to optimize
its configuration. This includes:

- Operating system type and version
- Available CPU cores and memory
- Installed runtime versions (Node.js, Python)
- Shell type for CLI integration

All system information is stored locally and never transmitted. You can
view the collected data at `~/.dojolm/system-profile.json`.

## Troubleshooting

If DojoLM fails to detect your system configuration, you can manually
specify values in the config file. Common environment variables like
PATH, HOME, and SHELL are used for standard path resolution only.
