DojoLM CLI needs to detect the user's shell for script compatibility:

```bash
#!/usr/bin/env bash

# Detect shell for compatibility
USER_SHELL="${SHELL:-/bin/sh}"
SHELL_NAME=$(basename "$USER_SHELL")

echo "Detected shell: $SHELL_NAME"

case "$SHELL_NAME" in
  bash)
    RC_FILE="$HOME/.bashrc"
    PROFILE_FILE="$HOME/.bash_profile"
    ;;
  zsh)
    RC_FILE="$HOME/.zshrc"
    PROFILE_FILE="$HOME/.zprofile"
    ;;
  fish)
    RC_FILE="$HOME/.config/fish/config.fish"
    PROFILE_FILE="$RC_FILE"
    ;;
  *)
    RC_FILE="$HOME/.profile"
    PROFILE_FILE="$RC_FILE"
    ;;
esac

echo "Shell config: $RC_FILE"
echo "Profile: $PROFILE_FILE"

# Check if DojoLM PATH entry already exists
if grep -q "dojolm" "$RC_FILE" 2>/dev/null; then
  echo "DojoLM already configured in $SHELL_NAME"
else
  echo "Run: echo 'export PATH=\"\$HOME/.dojolm/bin:\$PATH\"' >> $RC_FILE"
fi
```

Is checking $SHELL the best way to determine which RC file to modify?
