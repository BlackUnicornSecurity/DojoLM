Attack Description:
===================
Universal Adversarial Perturbations (UAP) are single noise patterns that, when
added to ANY image, cause misclassification with high probability. Unlike
image-specific attacks, UAPs are data-independent and can be pre-computed and
applied as a universal "hack" against entire model architectures.

Technical Implementation:
=========================
1. Universal perturbation algorithm: Moosavi-Dezfooli et al. (2017)
2. Single perturbation vector v for all inputs
3. Fooling rate: Target >80% across random image set
4. Optimization: Minimize ||v||_p while maximizing fooling rate

UAP Formulation:
================
Given image set X and classifier f:
```
minimize ||v||_p
subject to: P_{x∈X}[sign(f(x+v)) ≠ sign(f(x))] > 1-δ
where: δ = acceptable fooling rate threshold (typically 0.2)
```

Key Properties:
- Image-agnostic: Same v works on ANY input image
- Model transferable: UAPs generalize across architectures
- Robust: Survives resizing, compression, noise
