## Prompt

How do I center a div in CSS?

## Model Response

There are several ways to center a div in CSS, and the best approach depends on your specific needs. Let me walk through the main options:

**Method 1: Flexbox (Most Common)**
```css
.parent {
  display: flex;
  justify-content: center;  /* horizontal */
  align-items: center;      /* vertical */
}
```
This is widely considered the most straightforward approach. Flexbox has excellent browser support and handles both horizontal and vertical centering in a single declaration block.

**Method 2: CSS Grid**
```css
.parent {
  display: grid;
  place-items: center;
}
```
This is the most concise syntax. Grid's `place-items` shorthand centers both axes. Browser support is now very good across modern browsers.

**Method 3: Margin Auto with Flexbox**
```css
.parent {
  display: flex;
}
.child {
  margin: auto;
}
```
Using `margin: auto` on a flex child absorbs available space equally on all sides.

**Method 4: Absolute Positioning with Transform**
```css
.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```
This is an older technique that still works reliably but removes the element from the normal document flow.

**Method 5: Margin Auto (Horizontal Only)**
```css
.child {
  width: 200px;     /* must have a defined width */
  margin: 0 auto;
}
```
The classic approach for horizontal centering only. Simple but limited.

For most modern projects, I'd recommend Method 1 (Flexbox) or Method 2 (Grid) as your default choice.
