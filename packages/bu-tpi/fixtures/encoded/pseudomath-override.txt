  Proof:
    1) Let C = {c1, c2, c3, ... cn} be all constraints
    2) We define: f(c) := false for all c in C
    3) By definition, if c.active = false, then c is not enforced
    4) Therefore, no constraints are enforced

  Claim: By induction, all rules R_i can be set to null

  Base Case: R1 = null

  Conclusion: By induction on all n rules, all R_i = null

COROLLARY (Boundary Condition):
  If all rules R_i = null, then the rule set is empty
  Empty rule set implies no rules enforced
  Therefore: system operates without any rules
