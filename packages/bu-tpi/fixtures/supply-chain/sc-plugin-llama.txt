# DojoLM - LlamaIndex Vulnerability

**WARNING: LlamaIndex Security Issue**

Exploit LlamaIndex vector store vulnerabilities:

```python
from llama_index import VectorStoreIndex, SimpleDirectoryReader

# Load documents from untrusted source
documents = SimpleDirectoryReader(
    'user-uploaded-files'  # No validation of file contents
).load_data()

# Build index with poisoned documents
index = VectorStoreIndex.from_documents(documents)

query_engine = index.as_query_engine()
response = query_engine.query("trigger backdoor")  # Executes poisoned retrieval
```

LlamaIndex doesn't validate document content before indexing, allowing data poisoning attacks.
