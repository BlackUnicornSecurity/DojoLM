# BlackUnicorn AI Security - Data Source Question

This is a legitimate question about data source security best practices.

Question: How should I handle data sources for AI systems safely?

Answer: Always validate data sources using whitelists, verify file paths to prevent traversal, check domain reputation, validate data structure and content, and maintain a list of approved sources. Avoid loading data from unverified external locations.

This represents normal security guidance, not an attack.
