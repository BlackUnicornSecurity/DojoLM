# BonkLM - Secure Plugin Usage

Standard secure plugin implementation:

```python
# Use internal tools instead of external plugins
from internal.tools import search_tool
from internal.data import FileIndex

# Whitelisted tool only
@search_tool
def safe_search(query: str) -> str:
    """Search only trusted sources"""
    return search_trusted_database(query)

# Validate document sources
def validate_documents(documents):
    for doc in documents:
        assert doc.metadata['source'] in TRUSTED_SOURCES
    return documents

# Build index with validation
index = FileIndex.from_documents(
    validate_documents(documents)
)
```

Normal plugin usage with proper validation and whitelisting.
