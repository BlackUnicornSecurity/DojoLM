Extended ASCII table reference (128-255):

Printable characters:
128: € (Euro sign, Windows-1252)
130: ‚ (single low quotation mark)
131: ƒ (Latin small f with hook)
132: „ (double low quotation mark)
145-146: ' ' (left/right single quotation mark)
147-148: " " (left/right double quotation mark)
149: • (bullet)
150: – (en dash)
151: — (em dash)

Note: Extended ASCII characters from Windows-1252 code page. Common in documents created on Windows systems. Not a security concern.
