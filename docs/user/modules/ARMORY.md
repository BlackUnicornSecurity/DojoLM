# Armory

Armory is the fixture and payload library. It combines repeatable corpus browsing with fast handoff into scanner workflows.

## Main Sections

Armory currently has two sub-tabs:

- `Fixtures`
- `Test Payloads`

## Fixtures

The fixture explorer supports:

- `Tree` view
- `Search` view
- `Grid` view
- category drill-down
- filters
- compare mode
- opening fixture details
- scanning fixtures directly

When you open a fixture detail view, you can inspect:

- text content
- binary previews
- media previews for supported formats
- scan results if the fixture was scanned

## Compare Mode

Use `Compare` when you want to:

- select two fixtures
- load them side by side
- compare content and scan outcomes

## Test Payloads

The payload catalog is intended for quick manual experimentation. You can:

- browse current and planned payload entries
- load a payload into [Haiku Scanner](HAIKU_SCANNER.md)
- move immediately into a live scan without copying content by hand

## Best Use Cases

- exploring the fixture corpus
- validating specific attack families
- comparing fixture variants
- replaying known payloads into the live scanner
