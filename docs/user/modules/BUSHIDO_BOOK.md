# Bushido Book

Bushido Book is the compliance and framework-analysis module.

## What It Covers

- framework summaries
- coverage analysis
- gap review
- audit trail access
- checklists
- framework navigation
- framework-scoped compliance scans

## Layout

The module currently combines:

- a high-level score area
- a selectable framework list
- grouping by tier or by category
- a sub-view navigation bar

## Current Sub-Views

- `Overview`
- `Coverage`
- `Gap Matrix`
- `Audit Trail`
- `Checklists`
- `Navigator`
- `Framework Compliance`

## Framework List

The left-side framework area lets you:

- select one framework at a time
- browse by implementation tier or control category
- compare framework coverage at a glance

## Framework Compliance

Use `Framework Compliance` when you want a framework-specific testing or heatmap view tied to the chosen framework.

## LLM Testing Handoff

Bushido Book can hand you into LLM testing with the selected framework context, making it useful when you want to move from posture review into active execution.

## Best Use Cases

- reviewing coverage progress
- identifying framework gaps
- collecting audit context
- checking whether an LLM testing program maps cleanly to a specific framework
