# The Kumite

The Kumite is the strategic analysis hub. It groups multiple subsystems under one module instead of exposing them as separate top-level navigation items.

## Current Subsystems

- `SAGE`
- `Battle Arena`
- `Mitsuke`
- `Amaterasu DNA`
- `Kagami`
- `Shingan`

## How The Module Works

You start on an overview page with subsystem cards. Once you enter a subsystem, The Kumite switches to a tabbed view and exposes subsystem-specific onboarding and configuration drawers.

## SAGE

Use SAGE for:

- evolving adversarial prompts
- mutating seed prompts
- tracking fitness scores
- quarantining high-risk outputs

## Battle Arena

Use Battle Arena for:

- attacker versus defender matchups
- game-mode testing
- scoring and replay review

## Mitsuke

Use Mitsuke for:

- threat-feed ingestion
- alerting
- indicator extraction

## Amaterasu DNA

Use DNA for:

- lineage analysis
- family trees
- clustering and mutation relationships

## Kagami

Use Kagami for:

- mirror testing
- version comparison
- behavioral consistency review

## Shingan

Use Shingan for:

- deeper prompt-injection detection
- trust-boundary analysis
- supply-chain-oriented assessment

## Notes

- `Amaterasu DNA` is part of The Kumite, not a separate top-level module.
- The module includes onboarding steps and guide drawers for each subsystem.
