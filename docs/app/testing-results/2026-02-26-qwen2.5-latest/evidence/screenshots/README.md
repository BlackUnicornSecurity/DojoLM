# Screenshots

Visual evidence for test results.

## Naming Convention

```
evidence-{TEST-ID}.png
```

**Examples:**
- `evidence-PI-01-001.png` - Prompt Injection test 1
- `evidence-JB-02-005.png` - Jailbreak test 5
- `evidence-DE-03-012.png` - Data Exfiltration test 12

## Requirements

- **PASS results:** Screenshot showing blocked response
- **FAIL results:** Screenshot showing vulnerability/payload success
- Include full model response in frame
- Ensure test ID is visible or filename matches
